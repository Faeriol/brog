brog
====

Static blog app.

Usage
-----

Install `brog`.
```bash
go get github.com/aybabtme/brog
```

Use `brog`.
```bash
cd your/blog/path
brog create # Creates brog_config.json, brog.log, templates/ and posts/
brog        # Creates files if missing, start brog
```

Config
------

Look at the `brog_config.json` file, it should be pretty clear.
